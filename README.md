# ETL Data Pipeline

## 📌 Overview
This project demonstrates a Python-based ETL (Extract, Transform, Load) pipeline.
It extracts raw customer transaction data from CSV, transforms it by applying business rules (normalizing account numbers, validating PAN), and loads the cleaned data into Oracle 23c.
The pipeline showcases data engineering skills and integration with relational databases.

---

## 🛠️ Tech Stack
- **Language:** Python 3.14
- **Libraries:** pandas, oracledb, regex
- **Database:** Oracle 23c Free Edition
- **Tools:** Git, Postman, FastAPI (optional for API exposure)
